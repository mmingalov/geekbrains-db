ЗАДАНИЕ:
В базе данных Redis подберите коллекцию для подсчета посещений с определенных IP-адресов. 
РЕШЕНИЕ:
ip-адреса могут повторяться, поэтому  фиксировать посещения будет удобно в СПИСОК (коллекции и хэши здесь не подойдут, поскольку хэши подразумевают уникальные ключи, а коллекции подразумевают уникальные значения). Список будет содержать ip-адрес, данная запись будет фиксировать одно посещение, т.е. один элемент списка -- это одно посещение с определенного адреса.
пример пополнения списка ip приведен ниже
rpush ip "127.0.0.1" "127.0.0.2" "127.0.0.3" "127.0.0.1"
rpush ip "127.0.0.2" "127.0.0.3" "127.0.0.4" "127.0.0.5" "127.0.0.1"

команда llen ip вернет 9
