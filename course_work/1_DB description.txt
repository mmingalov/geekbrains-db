База данных хранит геологическую информацию о месторождении рудного золота.
Структура таблиц представлена следующим образом. Отступы показывают иерархию родительско-дочерних отношений (везде 1 ко многим).

	PROJECT
		SITE
			SITE_SURVEY
			LITHOLOGY
			SAMPLE*
			
		DESPATCH
			DESPATCH_DETAIL
			DESPATCH_SAMPLE*
			
			RECEIPT
				RESULT
			
	LKP_CATEGORY
		LKP_CODE
		

LKP_CATEGORY и LKP_CODE хранят справочные коды отдельных категорий данных (тип скважины, тип породы и т.п.)
Данные коды используются в таблицах БД.

PROJECT - месторождения/участки
SITE - скважины
SITE_SURVEY - координаты начала (устья) выработки
LITHOLOGY - геологическое описание интервалов скважины
SAMPLE - полевые геологические пробы, отобранные при бурении скважины
DESPATCH - реестр отправленных проб в лабораторию на анализ содержаний
DESPATCH_DETAIL - параметры запрошенных анализов в отправке
DESPATCH_SAMPLE - перечень проб, включенных в отправку
RECEIPT - протокол лаборатории о выполненных анализах проб
RESULT - содержания по каждой пробе из реестра отправки
